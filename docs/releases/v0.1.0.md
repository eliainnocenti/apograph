# Apograph v0.1.0 — first public beta

This prerelease contains one tested product: the unofficial PoliTo Beamer
presentation starter. Other directories in the repository remain draft
inventory and are not included as downloadable templates.

## Included

- A minimal `main.tex` starter with configuration isolated in `config.tex`.
- A separate `showcase.tex` component guide and release preview.
- Self-contained shared LaTeX modules at ZIP root for local, VS Code, and
  Overleaf use.
- Deterministic ZIP, SHA-256 sidecar, build report, catalog snapshot, and
  release index.
- Compiling box/text fallbacks when institution branding is absent.

## Known limitations

- This is an unofficial beta and is not endorsed by Politecnico di Torino.
- The distributed artifact contains no PoliTo logos or branded backgrounds.
  Users may add assets they are independently authorized to use at the paths
  documented in the template README.
- The theme has been tested with pdfLaTeX in TeX Live 2025 on Overleaf and TeX
  Live 2026 in CI. Other engines and older distributions are not part of this
  release's compatibility claim.
- Public theme commands and layout details may change before a stable release.

Report reproducible problems through [Apograph Issues](https://github.com/eliainnocenti/apograph/issues).
Include the template ID, release version, compiler/environment, and the smallest
source that demonstrates the problem.
